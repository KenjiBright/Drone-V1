# Python Script - Cách sử dụng chi tiết

Hướng dẫn từng bước sử dụng Python script để capture dữ liệu Black Box từ Drone V1.

---

## 🔧 Cài đặt

### 1. Cài đặt Python (nếu chưa có)
- Tải từ: https://www.python.org/downloads/
- Cài đặt, **nhớ tick "Add Python to PATH"**

### 2. Cài đặt Dependencies

**Windows (Command Prompt):**
```
cd d:\Code C-C++\Drone V1\tools
pip install -r requirements.txt
```

**Linux/MacOS (Terminal):**
```
cd ~/drone-v1/tools
pip3 install -r requirements.txt
```

### 3. Kiểm tra cài đặt

```
python --version          # Windows
python3 --version         # Linux/Mac
```

---

## 🚁 Sử dụng Script

### Cách 1: Command Line (dễ nhất)

#### Windows

```bash
# Cách nhanh - double-click vào capture.bat
capture.bat

# Hoặc command line với tham số:
python capture_blackbox.py --port COM17 --duration 30 --desc "test1"
```

**Tìm COM port:**
1. Kết nối ESP32 → Device Manager (Y tổ hợp phím Win+X)
2. Tìm "Ports (COM & LPT)" → xem COM port (ví dụ: COM17)
3. Thay vào `--port` parameter

#### Linux/MacOS

```bash
# Chạy script shell
chmod +x capture.sh
./capture.sh

# Hoặc Python trực tiếp
python3 capture_blackbox.py --port /dev/ttyUSB0 --duration 30 --desc "test1"
```

**Tìm serial port:**
```bash
ls /dev/tty*
# Thường là /dev/ttyUSB0 hoặc /dev/ttyACM0
```

### Cách 2: Phân tích file CSV cũ

```bash
# Windows
python capture_blackbox.py --analyze flight_20260412_143630_test1.csv

# Linux/Mac
python3 capture_blackbox.py --analyze flight_20260412_143630_test1.csv
```

### Cách 3: Chỉ capture, không vẽ biểu đồ

```bash
python capture_blackbox.py --port COM17 --duration 30 --desc "test1" --no-plot
```

## 📊 Quy trình Capture Chi Tiết

### Bước 1: Chuẩn bị drone

```
1. Kết nối ESP32 → USB → Computer
2. Serial Monitor hiển thị menu
3. Đặt drone trên bàn (không chân quạt)
4. Gạt ARM switch xuống (DISARM)
5. Throttle stick xuống dưới cùng
```

5. Bay drone ~30 giây
6. Nhấn E để dừng

### Biểu đồ PNG

Tên: `flight_20260412_143630_hover_test.png`

Hiển thị:
- **Row 1:** Roll Response (Setpoint vs Actual) + Roll PID Components
- **Row 2:** Pitch Response + Pitch PID Components
- **Row 3:** Yaw Response + Yaw PID Components

**Cách đọc:**
- Blue = Setpoint (lệnh từ RC)
- Red = Actual (sensor gyro)
- Khoảng cách nhỏ = PID ổn
- Khoảng cách lớn = Kp quá thấp
- Vượt quá = Kd quá thấp/ Kp quá cao

## 🐛 Troubleshooting

| Lỗi | Nguyên nhân | Giải pháp |
|---|---|---|
| `ModuleNotFoundError: No module named 'serial'` | pyserial chưa cài | `pip install pyserial` |
| `ModuleNotFoundError: No module named 'pandas'` | pandas chưa cài | `pip install pandas matplotlib` |
| `Serial port COM17 not found` | COM port sai | Kiểm tra Device Manager, thay port |
| `Timeout waiting for CSV header` | Drone không gửi data | Nhấn 1 trên drone menu để bắt đầu log |
